📝 Mi Blog de Notas (Proyecto Final)
¡Hola! Este es el frontend de mi proyecto final. Es una aplicación sencilla pero con mucho estilo para anotar tareas y no olvidar nada. La idea era que se viera bien, fuera fácil de usar y que funcionara rápido.

🚀 ¿Qué hace?
Lista de Tareas: Se conecta a una base de datos para mostrar lo que hay que hacer.

Modo Oscuro: Tiene un botón de luna 🌙 para que no te queden los ojos rojos de noche.

Letras a tu gusto: Puedes agrandar o achicar el texto con los botones A+ y A-.

Diseño Limpio: Sin menús raros ni cosas que estorben.

🛠️ ¿Qué usé?
Angular 17 (lo último de lo último).

CSS para que todo se vea oscuro y elegante.

Express en el backend para manejar los datos.

⚙️ ¿Cómo lo pruebas?
Clonas esto.

Haces un npm install (esperas un ratico).

Tiras un ng serve.

Vas a localhost:4200 y ¡listo!

Nota: No olvides prender el backend primero para que la tabla no te salga vacía. 😉
